# Customer Churn Prediction - Capstone Project

## Project Overview
This project builds a machine learning model to predict customer churn 
for a telecommunications company. The goal is to identify customers who 
are at high risk of canceling their service so the retention team can 
take proactive action before the customer leaves.

## Business Problem
The company is currently experiencing a 25% customer churn rate, 
resulting in approximately $175,000 in lost monthly revenue. This project 
uses historical customer data to predict which customers are likely to 
churn and flags anyone with a probability score above 60% as high risk.

## Dataset
IBM Telco Customer Churn dataset
Source: Kaggle (blastchar/telco-customer-churn)
7,043 customer records and 21 columns
